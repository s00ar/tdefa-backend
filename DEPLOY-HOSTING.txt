Contenido de entrega:

- `public/`: frontend compilado listo para subir
- `server/`: API Node que atiende `/api`
- `package.json` y `package-lock.json`: dependencias del backend

Notas:

- La app ya apunta a `/api` en hosting.
- Si usas Apache/cPanel, la build incluye `.htaccess` para SPA routing.
- Esta app no es solo estatica: para que partidos/planillas funcionen, el hosting debe soportar Node.js y MySQL.
- Variables esperadas por la API: `DB_HOST`, `DB_PORT`, `DB_USER`, `DB_PASSWORD`, `DB_NAME`, `API_PORT`.
